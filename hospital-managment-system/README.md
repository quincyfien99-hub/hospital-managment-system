# Hospital Management System (HMS)

A comprehensive Hospital Management System built with Node.js, Express, MySQL, and vanilla HTML/CSS/JavaScript.

## Features

- **User Authentication**: Login/Register with JWT tokens
- **Role-Based Access Control**: Admin, Doctor, Nurse, Staff, Patient
- **Patient Management**: CRUD operations for patient records
- **Appointment Scheduling**: Book, view, and manage appointments
- **Responsive Design**: Mobile-friendly interface
- **Demo Mode**: Works without database setup

## Tech Stack

### Backend
- Node.js + Express.js
- MySQL Database
- JWT Authentication
- Bcrypt for password hashing

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap 5 for styling
- Fetch API for HTTP requests
- LocalStorage for session management
